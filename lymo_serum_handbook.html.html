Part 01 · 配方逻辑

先搞懂
这瓶产品

大多数精华的逻辑是堆成分。这瓶不一样——
它按照皮肤真正的接收顺序来设计：

一
先让皮肤平静
主播这样理解

皮肤就像一道门。屏障受损时门是破的——外面刺激进来，里面水分流失，涂再贵的成分也留不住。
第一件事：先把破损的门修好。

乳酸杆菌发酵产物 ← 成分表第一位，不是水
发酵小分子从底层修护皮肤微生态，让皮肤恢复健康接收状态
积雪草 + 黄花蒿 + 甘草
三个植物提取物专门处理炎症反应。晒后红热刺痛，先把火扑灭
丁二醇 + 戊二醇
溶剂体系，配合发酵小分子提升渗透效率——这就是为什么用完吸收快
✓ 晒后修复 ✓ 好吸收
二
让皮肤干净
主播这样理解

屏障修好之后，清理垃圾。
自由基让皮肤氧化变暗 → 这是提亮的根本
堆积皮脂和角质堵毛孔 → 这是缩毛孔的根本

五重抗氧化：烟酰胺 · 松口蘑 · 倒捻子 · 迷迭香 · 黄花蒿
五个成分从不同路径清除自由基。单一成分只能堵一条路，多通道同时堵，氧化才真的被拦住

三
让皮肤发光
主播这样理解

前两步解决今天的问题。
这一步解决时间的问题——让皮肤结构越来越好，效果持续往好的方向走。

棕榈酰三肽-1 + 棕榈酰四肽-7（Matrixyl 3000）
三肽促进胶原蛋白合成，四肽抑制炎症因子，两个肽协同让皮肤结构越来越好
银耳 + 松口蘑 + 药用层孔菌
东方来源真菌发酵提取物，保湿、修护、细胞更新，长效发挥
✓ 长效改善 ✓ 持续变好

Part 03 · 直播话术

四个痛点
怎么讲

说问题 → 说原因 → 说这瓶怎么解决

01
晒后修复
夏季主推 · 场景感强
+
夏天最崩溃的事是什么？出门半小时回来，脸又红又热又刺，敷面膜也只是表面凉快，第二天还是暗。

为什么？因为晒后的皮肤是轻度炎症状态，紫外线破坏了皮肤细胞，面膜只能补水，处理不了炎症。

这瓶精华里的积雪草、黄花蒿、甘草，三个成分专门处理炎症反应，不只是让皮肤凉下来，是从细胞层面开始修复。

第三方检测：30名敏感肌受试者使用7天，皮肤泛红改善超过20%——仪器测出来的，不是主观感受。
02
缩小毛孔
搜索量大 · 共鸣度高
+
毛孔粗大是天生的吗？不完全是。毛孔看起来大，90%是两个原因：油脂太多把毛孔撑大了，或者角质堆积堵在毛孔口。

水杨酸能清除堆积，短期有效，但没有从源头调节皮脂分泌和代谢节奏，需要持续使用才能维持。

这瓶用的是烟酰胺加人参皂苷：烟酰胺从源头调节皮脂，油少了毛孔不被撑；人参皂苷促进角质代谢，毛孔口不堵了自然就小了。

改变的是皮肤的代谢状态，效果更稳定。
03
提亮肤色
反常识切入 · 记忆点强
+
用了很久美白产品，脸还是暗？不一定是产品骗人，可能是搞错了问题。

脸黑和脸暗是两件事：脸黑是色素问题，美白产品有用；脸暗是氧化问题，美白产品没用。整张脸均匀发灰发黄、没有透亮感，基本上就是氧化了。

这瓶用了五个成分多通道抗氧化。其中松口蘑和倒捻子，生产企业有国家授权的发明专利——不是买来的通用原料，是自主研发的技术。

皮肤干净了，光自然透出来——这种亮不是反光，是真的透亮。
04
好吸收
使用体验 · 建立信任
+
很多精华涂上去黏黏的，半天吸收不了——不是你皮肤的问题，是配方的问题。

很多品牌加很多增稠剂让质地看起来饱满，结果成分堆在皮肤表面进不去。

这瓶成分表第一位不是水，是乳酸杆菌发酵产物。发酵小分子渗透性天然强，配合溶剂体系让整瓶精华吸收效率都提高了。

检测数据：用完4小时皮肤保湿力提升74%——涂上去快不是因为没营养，是因为东西真的进去了。

Part 06 · 常见问题

用户问了
怎么答

和其他精华有什么不一样？ +
别人堆成分，我们想的是顺序。先修屏障，再清理，再修护。皮肤稳定了，成分才真的有用。而且核心成分有两项国家发明专利，不是行业通用原料。
敏感肌可以用吗？ +
可以。第三方检测的30名受试者本身就是敏感肌人群（乳酸刺痛阳性），全程无不良事件。但急性过敏期——红肿脱皮大面积刺痛——先就医，稳定后再用。
多久能见效？ +
吸收好、晒后舒缓，当天有感觉。提亮和毛孔，两到四周开始有变化。检测数据显示，连续28天各项指标持续改善，100%受试者想继续使用。
有没有检测报告？ +
有。广东立创检测技术服务有限公司出具的第三方人体功效测试报告，30名敏感肌受试者，7天和28天双维度检测，结果均有统计显著性（P＜0.05）。
男生可以用吗？ +
可以。抗氧化、修护、缩毛孔，男生也有这些需求，成分没有性别限制。
